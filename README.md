# Drone Intec Mockup

Mockup vitrine statique realise pour presenter une nouvelle direction visuelle du site Drone Intec.

## Contenu

- `index.html` : page principale
- `styles.css` : styles du mockup
- `main.js` : interactions frontend
- `assets/` : images et visuels utilises dans la maquette

## Lancer en local

Le mockup peut etre servi avec n'importe quel serveur statique.

Exemple Python :

```bash
python -m http.server 4173
```

Puis ouvrir :

```text
http://127.0.0.1:4173/
```
